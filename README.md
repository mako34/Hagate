# Budgie

A simple Hello World VS Code extension.

## Features

- Adds a "Hello World" command to the command palette

## Usage

1 compile
npm run compile


2 open vscode using the extension
code --extensionDevelopmentPath="/Users/manuel.betancurt/Library/CloudStorage/MaDrive/Documents/MK/testeo/vsX/budgie" "/Users/manuel.betancurt/Documents/GyG/bhyve-feature-user"


Action	Shortcut mac
Start	⌘ + ⌥ + B (Cmd + Option + B)
Stop	⌘ + ⌥ + S (Cmd + Option + S)

Action	Windows/Linux	Mac
Start	Ctrl+Alt+B	Cmd+Alt+B
Stop	Ctrl+Alt+S
